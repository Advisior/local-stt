# Support

## Getting Help

- **Bug Reports:** [Open an issue](https://github.com/Advisior/local-stt/issues/new?template=bug.md)
- **Feature Requests:** [Open an issue](https://github.com/Advisior/local-stt/issues/new?template=feature.md)
- **Security:** See [SECURITY.md](SECURITY.md)

## Troubleshooting

Before opening an issue, check the [Troubleshooting section](README.md#troubleshooting) in the README.

When reporting a bug, please include:
- A clear description of the problem
- Steps to reproduce
- OS and Python version
- Relevant logs (set `CLAUDE_STT_LOG_LEVEL=DEBUG`)

## Response Times

This is a community-maintained project. We aim to respond to issues within 7 days.
